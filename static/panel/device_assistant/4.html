Attribute: Toggles

Enable it if the hardware can not confirm its status after execute a command.

Toggles are like switches. It could be on or off.

For example in a washer machine it could exist a toggle for enabling fast centrifugation. In this case we create a toogle named Fast centrifugation and we select en has the word's languaje.

It is possible that we use differents words for the same toggle: we can say Ok Google, enable Fast centrifugation at the washer machine or we can say Ok Google, enable Heavy centrifugation at the washer machine.

Heavy centrifugation will be a synonym for the toggle's name. We wrtie the synonyms down into the Synonymous text box separated by commas.

Attribute: ArmDisarm

A security system can have differents levels of status, for example, global arm or parcial arm. Imagine that parcial arm only enables the garage's sensors

Write the main level's name down into the Name texto box. For example, parcial arm.

Select the languaje.

Write all possibles synonyms of the main name separated by commas. For example, garage only, incomplete arm.

Add every level using the add button.

Attribute: Cook

Food presets

Cooking modes

This devices usually have preprogrammed recipe. Food presets represent these recipe.

Name is the main name for the receipt, for example Cup of chocolate.

Units is the units in with the name is meseaured. In this case there is no units so we write NO_UNITS down.

Lang is the languaje.

Synonym are the other possibles ways of calling the receipt. For example, Glass of chocolate,chocolate,mug of chocolate.

Attribute: Rotation

These represent the minimum and maximum degrees that a device can rotate. Must be positive values.
"Indicates if the device can only be queried for state information, and cannot be controlled."
Set to true if the device allows continuous rotation. For example, a device currently at 270 degrees, rotated by 100 degrees clockwise, would go to 10 degrees.
Set to true if the device allows rotation by degree.
Set to true if device allows rotation by percent. Note that one device may support either or both of these attributes (degrees and percent).
